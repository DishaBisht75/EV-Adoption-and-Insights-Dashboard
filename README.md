#  Electric Vehicle Adoption & Insights Dashboard

##  Project Overview

This Power BI project analyzes Electric Vehicle (EV) adoption trends using interactive dashboards and data visualization techniques. The dashboard provides insights into EV distribution, vehicle make and model popularity, electric range performance, CAFV (Clean Alternative Fuel Vehicle) eligibility, geographic trends, utility provider coverage, and policy-related factors.

The project aims to support data-driven decision-making for sustainable transportation planning, EV infrastructure development, and policy evaluation.

---

##  Objectives

- Analyze EV adoption trends across different model years.
- Identify the most popular EV makes and models.
- Examine EV distribution across counties and cities.
- Evaluate CAFV eligibility patterns.
- Analyze electric range performance of different EV models.
- Explore utility provider coverage for EV users.
- Support infrastructure planning and policy-related insights.

##  Key Insights

- Tesla dominates EV adoption among all vehicle manufacturers.
- Battery Electric Vehicles (BEVs) account for the majority of EV registrations.
- King County shows the highest concentration of EV adoption.
- CAFV eligibility varies significantly across vehicle categories.
- Utility provider distribution highlights potential infrastructure demand areas.
